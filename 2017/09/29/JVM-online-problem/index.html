JVM 线上问题快速排查

Sep 29, 2017 | Java

找到最耗CPU的线程

步骤一:找到最耗CPU的进程

  • 执行top -c, 显示进程运行信息列表
  • 输入 P(大写p), 进程按照CPU使用率排序

步骤二:找到最耗CPU的线程

  • top -Hp pid,显示一个进程的线程运行信息列表
  • 输入 P(大写p), 进程按照CPU使用率排序

步骤三:将线程PID转化为16进制

之所以要转化为16进制，是因为堆栈里，线程id是用16进制表示的

步骤四:查看堆栈，找到线程在干嘛

找到最耗内存对象

  • #instances 实例数
  • #bytes 所占内存大小
  • class name 类名
